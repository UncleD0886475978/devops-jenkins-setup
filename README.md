# DevOps Jenkins CI/CD Setup

A complete Jenkins CI/CD server setup on RHEL 10 VM.

## What's Included
- RHEL 10 VM network configuration (VMware vmxnet3 bridged adapter fix)
- Jenkins 2.555 installation and setup
- Java 21 OpenJDK
- Git 2.47
- Python CI/CD lab series (Flask app, Docker, Kubernetes, Webhooks)

## Lab Series
- Lab 1: First Jenkins Pipeline with Python/Flask
- Lab 2: Docker Build & Registry Push
- Lab 3: Git Webhooks & Auto-Trigger
- Lab 4: Kubernetes Deployment (Full CD)

## Tech Stack
- OS: Red Hat Enterprise Linux 10.1
- Jenkins: 2.555.1
- Java: OpenJDK 21
- Git: 2.47.3
- Python: Flask, pytest
